# QuickBooks Cleanup Sample Project

## Project Flow

1. Raw transaction data (raw-data/)
2. Issues identified (issues-identified/)
3. Cleanup actions performed
4. Final summary (final-summary/)
5. Before vs After financial reports

This project demonstrates a full cleanup process from raw data to corrected financial statements.

## Overview
This project demonstrates a full bookkeeping cleanup for a small service business covering Jan–Mar 2025.

---

## Initial Issues Found
- Uncategorized transactions
- Duplicate income entries
- No bank reconciliation
- Personal expenses mixed with business expenses
- Disorganized chart of accounts

---

## Actions Taken
- Categorized all transactions properly
- Removed duplicate entries
- Performed full bank and credit card reconciliation
- Separated personal and business expenses
- Restructured chart of accounts

---

## Results
- Clean and reliable financial reports
- Fully reconciled accounts
- Improved financial visibility

---

## Scope of Work
- Bank reconciliation
- Expense categorization
- Journal entries and adjustments
- Financial report preparation

---

## Notes
All data used in this project is simulated for demonstration purposes.
